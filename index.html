Recruitify

  • Recruitify is a MERN Stack based web application.
  • Recruitify provides an optimized Interview Platform for recruitements of new Employees/students.
  • Features include video calling(using WebRTC) for the interview.
  • Real-time live notepad(using Socket.io).
  • View past and pending interviews.
  • Mass-mailer to send links to the Employees/students for interviews.
  • Technologies used- NodeJS, ReactJS, MongoDB,WebRTC.
